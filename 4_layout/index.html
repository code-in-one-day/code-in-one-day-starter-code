Block Elements

This is a H3 element, a block element. All header elements, h1 through h6, are block level elements.

This is a p tag, also a block element. Block elements occupy the whole line, and always start on a new line.

Block level elements have both height and width.

Inline Elements

An anchor tag is an example of an element. This is a span element, an inline element. Inline elements take up only as much width as necessary, and do not start on new lines. This is a code element, an inline element. Inline elements only have as much height and width as necessary, and these cannot be modified.

Inline Block Elements

Inline block elements have both height and width. However, they do not start on new lines. This makes them useful for putting things with height and width next to each other.
